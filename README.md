# Cloud storage app

## Функционал приложения

### Работа с пользователями:
- Регистрация
- Авторизация
- Logout
### Работа с файлами и папками:
- Загрузка файлов и папок
- Создание новой пустой папки (аналогично созданию новой папки в проводнике)
- Удаление
- Переименование
